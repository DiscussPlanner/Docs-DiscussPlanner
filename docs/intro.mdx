# Introduction

## Quick Start
フロントエンド用サーバーとバックエンドサーバーを起動させる必要があります。

1. 該当ディレクトリに移動

    ```powershell
    cd dev
    ```

2. サーバーの起動

    ```powershell
    docker-compose up
    ```

4. ブラウザでアクセス

    [http://localhost:80](http://localhost:80) にアクセスする事でアプリケーションサーバーにアクセスできます。

## サーバーの停止
以下のコマンドを実行し、サーバーを停止させてください。

```powershell
docker-compose down --rmi all
```
