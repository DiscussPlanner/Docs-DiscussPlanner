# プライバシーポリシー

当アプリケーション（以下、「本アプリ」）をご利用いただくにあたり、ユーザーのプライバシーを保護することを重要視しています。本プライバシーポリシーでは、どのような情報を収集し、どのように使用するかについて説明します。

## 1. 収集する情報

### 1.1 ユーザー提供情報
本アプリを利用する際、以下の情報を提供していただく場合があります：
- Notionユーザー名
- メールアドレス
- ワークステーション情報

### 1.2 自動収集情報
本アプリは以下の情報を自動的に収集することがあります：
- Notion API から取得した情報

## 2. 情報の利用目的

収集した情報は以下の目的で利用されます：
- 利用状況の分析によるサービス改善
- 必要に応じた法的義務の遵守

## 3. 情報の共有・第三者提供

本アプリは、収集した情報を第三者と共有することはありません。ただし、以下の場合を除きます：
- ユーザーの同意を得た場合
- 法律や規制に基づく要求がある場合

## 4. 情報の保存期間

ユーザーの情報は、利用目的が達成され次第、適切な方法で削除されます。ただし、法令で定められた保存期間がある場合はその期間保持します。

## 5. ユーザーの権利

ユーザーは以下の権利を有します：
- 自分の情報へのアクセス
- 情報の修正または削除の要求
- データ処理の制限または異議申し立て

## 6. セキュリティ

本アプリは、ユーザー情報を保護するために適切なセキュリティ対策を講じています。ただし、100%の安全性を保証するものではありません。

## 7. 本ポリシーの変更

本プライバシーポリシーは、必要に応じて改訂されることがあります。改訂内容は本ドキュメント内で公開されます。
